# Favourite Student List

A simple React application for managing a list of favourite students. This project demonstrates React Router and the useContext hook for global state management.

## Features

- View the student list
- Add students to favourites
- Prevent duplicate favourites
- Remove students from favourites
- Navigate between pages using React Router
- Dynamic UI updates

## Technologies Used

- React JS
- React Router
- useContext
- useState
- CSS
- Vite

## How to Run

1. Install dependencies

   npm install

2. Start the development server

   npm run dev

## Project Purpose

This project was created to understand routing, global state management using Context API, and conditional rendering in React.
